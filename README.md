
# Garage Management System

Backend is running on http://localhost:3000

Frontend is running on http://localhost:5173/




#### To install dependencies

```http
  npm install
```

#### To start backend

```http
  npm start
```


#### To start frontend

```http
  npm run dev
```




## Tech Stack

**Client:** React, TypeScript, Redux, TailwindCSS

**Server:** Node, Express
